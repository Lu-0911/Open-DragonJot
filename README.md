# DragonJot - 点睛AI开源版

国内首个专注于舞龙运动的AI智能评判与训练辅助系统，基于计算机视觉技术实现对人体姿态与龙形轨迹的同步识别与分析。

## 项目简介

点睛AI（DragonJot）创新提出"人龙一体"协同动作识别模型，针对传统舞龙赛事评判主观性强、训练反馈缺乏量化依据等行业痛点。系统支持实时动作捕捉、动作分类与量化评估，覆盖赛事评判、专业训练与大众教学三大场景。

Open DragonJot作为开源版本，提供核心功能体验，包括：

- 人体姿态实时识别
- 龙骨架关键点追踪
- 舞龙动作自动分类
- 多源视频输入支持（本地文件/摄像头）

## 核心功能

1. **实时检测**：同步识别人体与龙骨架关键点
2. **动作分析**：基于深度学习模型的舞龙动作分类
3. **多模式支持**：图片/视频/摄像头实时处理
4. **可视化展示**：关键点连线与轨迹追踪可视化

## 快速开始

### 环境要求

- Python 3.8+
- 支持Windows/macOS/Linux系统
- 推荐GPU加速（NVIDIA显卡需支持CUDA）

### 从零开始运行指南

1. 克隆本项目到本地

   ```bash
   git clone https://github.com/yourusername/dragonjot.git
   cd dragonjot
   ```

2. 运行环境配置脚本（自动安装 Python、依赖包并启动程序）
   - Windows 系统：双击运行 `start_windows.bat`
   - macOS/Linux 系统：终端执行 `bash start_terminal.sh`

3. 浏览器将自动打开系统界面，默认地址：`http://localhost:8501`

4. 模型文件请联系项目开发人员获取

### 功能使用

- **图片模式**：上传舞龙图片，系统自动识别关键点
- **视频模式**：上传视频文件，生成带关键点标注的处理结果
- **摄像头模式**：实时捕捉摄像头画面并进行分析
